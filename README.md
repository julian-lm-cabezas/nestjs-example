### Installation

```
npm install
```

### Compile project locally
```
npm start
```
*server running on http://localhost:3000

### Test

```bash
# unit tests
$ npm run test
# e2e tests
$ npm run test:e2e
# test coverage
$ npm run test:cov
```

### Lints and fixes files
```
npm run lint
```
